# 境外采集器独立于境内网站并保持无状态

境外采集器作为独立仓库由 GitHub Actions 启动，只抓取获准公开来源、生成稳定签名批次并发送到境内，不运行 LLM、不保存数据库状态，也不依赖 Vault 网站代码。境内网站所在 VPS 负责批次持久化、异步 LLM 处理、事件编排和公开发布；这一拆分接受两个部署单元的维护成本，以换取境外任务足够轻量、可替换，以及用户数据和内容判断全部留在境内。

ADR-0004 保留本决策的独立、无状态和境内处理原则，并把原“每日四次、两类批次”修订为每小时统一调度的单一 `AcquisitionBatch` 管线。
